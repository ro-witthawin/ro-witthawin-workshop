# AI Workshops Hub

**A centralized home for hands-on AI workshops, labs, and tutorials built for real-world learning.**

This repository brings together production-minded examples, notebooks, reference architectures, and guided exercises for developers, students, and researchers exploring modern AI systems. The content spans **Beginner** to **Advanced** levels, with a strong focus on Python, PyTorch, Local LLMs, Agentic AI, and Computer Vision.

Whether you are prototyping your first RAG pipeline, deploying a local model server, or building multi-agent workflows, this repo is designed to help you move from concepts to working systems quickly.
